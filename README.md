# MediVzla - Sistema Médico Profesional

## 📌 Requisitos
- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Conexión a internet (solo para primer carga - funciona offline después)

## 🚀 Despliegue en Netlify
1. Clona el repositorio
2. Conéctalo con Netlify
3. El sitio se desplegará automáticamente

## 📚 Estructura de Archivos
- `src/css/` - Archivos CSS
- `src/js/` - Archivos JavaScript
- `src/pages/` - Páginas HTML
- `assets/` - Recursos estáticos

## 🧪 Notas
- Sistema completamente funcional con almacenamiento local
- Listo para producción
- Optimizado para despliegue en Netlify
